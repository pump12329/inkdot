# 思维导图设计规范

## 概述

本文档定义了InkDot思维导图的设计规范，遵循极简主义美学和现代交互设计原则，为用户提供专业、直观、美观的思维导图编辑体验。

## 设计原则

### 极简主义美学

- **少即是多**: 去除不必要的视觉元素，专注核心功能
- **纯净色彩**: 以黑白灰为主色调，避免过多色彩干扰
- **留白充足**: 充分利用空白空间，避免视觉拥挤
- **一致性**: 统一的视觉语言和交互模式

### 用户为中心

- **直观操作**: 操作方式符合用户直觉
- **快速响应**: 操作反馈及时，性能流畅
- **容错设计**: 允许用户犯错并提供恢复机制
- **渐进披露**: 核心功能优先，高级功能按需显示

## 视觉设计规范

### 节点设计

#### 基础样式

- **形状**: 圆角矩形，圆角半径8px
- **背景**: 默认透明背景，选中时白色背景
- **边框**: 细边框 1px，颜色 #00000022
- **内边距**: 上下8px，左右16px
- **字体**: 系统字体栈，14px大小，400字重

#### 状态设计

```
默认状态:
- 背景: transparent
- 边框: 1px solid #00000022
- 文字: #000000

悬停状态:
- 背景: #f8f9fa
- 边框: 1px solid #00000044
- 文字: #000000

选中状态:
- 背景: #ffffff
- 边框: 2px solid #000000
- 文字: #000000
- 阴影: 0 4px 6px rgba(0, 0, 0, 0.1)

编辑状态:
- 背景: #ffffff
- 边框: 2px solid #000000
- 输入框样式，聚焦指示器
```

#### 层级区分

- **主节点**: 字体16px，字重600，略大尺寸
- **一级子节点**: 字体14px，字重500，标准尺寸
- **二级子节点**: 字体14px，字重400，标准尺寸
- **三级及以下**: 字体12px，字重400，颜色#6c757d

### 连接线设计

#### 基础规范

- **样式**: 简洁直线或贝塞尔曲线
- **粗细**: 统一2px
- **颜色**: 半透明黑色 #00000044
- **无箭头**: 完全去除箭头装饰，保持线条纯净

#### 连接类型

```
默认连接:
- 粗细: 2px
- 颜色: #00000044
- 样式: 实线

重要连接:
- 粗细: 3px
- 颜色: #00000066
- 样式: 实线

临时连接:
- 粗细: 1px
- 颜色: #00000033
- 样式: 虚线

选中连接:
- 粗细: 3px
- 颜色: #000000
- 样式: 实线
```

#### 连接路径

- **直线连接**: 适用于简单布局
- **贝塞尔曲线**: 适用于复杂布局，避免重叠
- **智能路径**: 自动选择最佳路径避免穿越其他节点
- **端点吸附**: 连接端点精确吸附到节点边缘中心

### 画布设计

#### 背景选项

```
纯色背景 (默认):
- 颜色: #ffffff
- 无任何装饰

网格背景 (可选):
- 基础颜色: #ffffff
- 网格线: #00000008
- 网格大小: 20px × 20px
- 透明度可调节
```

#### 视觉边界

- **无限画布**: 理论上无限大的编辑空间
- **视觉边界**: 通过淡出效果暗示边界
- **中心指示**: 可选的中心十字线辅助定位

## 交互设计规范

### 节点操作

#### 基础交互

```
单击: 选择节点
双击: 进入编辑模式
拖拽: 移动节点位置
右键: 显示上下文菜单

键盘操作:
- Enter: 创建同级节点
- Tab: 创建子节点
- Delete: 删除选中节点
- F2: 重命名节点
- Esc: 取消当前操作
```

#### 多选操作

- **框选**: 拖拽矩形框选择多个节点
- **Ctrl+点击**: 累加选择节点
- **Shift+点击**: 连续选择节点
- **Ctrl+A**: 全选所有节点

### 连接操作

#### 创建连接

```
拖拽连接:
1. 从节点边缘开始拖拽
2. 显示预览连接线
3. 拖拽到目标节点
4. 释放鼠标完成连接

连接模式:
1. 激活连接工具
2. 点击起始节点
3. 点击目标节点
4. 自动创建连接
```

#### 连接编辑

- **选择连接**: 点击连接线选择
- **删除连接**: 选中后按Delete键
- **重新连接**: 拖拽端点到新位置
- **路径调整**: 拖拽控制点调整曲线

### 画布导航

#### 视图控制

```
平移画布:
- 空白区域拖拽
- 空格键 + 拖拽
- 中键拖拽

缩放控制:
- 鼠标滚轮
- Ctrl + 滚轮 (精确缩放)
- Ctrl + / Ctrl - (快捷键)
- 缩放工具按钮

视图快捷操作:
- Ctrl + 0: 重置缩放 (100%)
- Ctrl + Shift + F: 适应所有内容
- 双击空白: 重置视图位置
```

#### 导航辅助

- **小地图**: 右下角缩略图导航
- **缩放显示**: 当前缩放百分比显示
- **坐标指示**: 鼠标位置坐标显示（可选）

## 性能设计规范

### 渲染优化

#### 视口裁剪

```
实现策略:
- 只渲染可见区域节点
- 屏幕外节点延迟渲染
- 动态加载显示内容
- 智能预加载临近区域
```

#### 层次渲染

```
详细级别:
- 近距离: 完整渲染所有细节
- 中距离: 简化字体和装饰
- 远距离: 仅显示节点轮廓
- 极远距离: 点状显示
```

### 交互性能

#### 响应时间目标

```
操作响应:
- 点击反馈: < 50ms
- 拖拽跟随: < 16ms (60fps)
- 缩放响应: < 100ms
- 节点创建: < 200ms
```

#### 大量节点优化

```
处理策略:
- 节点数量 < 100: 正常渲染
- 节点数量 100-500: 简化连接线
- 节点数量 500-1000: 层次加载
- 节点数量 > 1000: 分区渲染
```

## 可访问性设计

### 键盘导航

```
导航支持:
- Tab: 在节点间循环导航
- 方向键: 选择相邻节点
- Enter: 编辑选中节点
- Esc: 退出编辑模式
- Delete: 删除选中节点
```

### 屏幕阅读器支持

```
语义标记:
- 节点角色: treeitem
- 画布角色: tree
- 工具栏角色: toolbar
- 状态通知: aria-live
```

### 视觉辅助

```
对比度要求:
- 文字对比度: ≥ 4.5:1
- 边框对比度: ≥ 3:1
- 状态指示清晰
- 支持高对比度模式
```

## 动画设计规范

### 过渡动画

```
节点动画:
- 出现: fade-in + scale(0.8 → 1.0)
- 消失: fade-out + scale(1.0 → 0.8)
- 移动: transform位置变化
- 状态: 200ms ease-out

连接动画:
- 创建: 描边动画从起点到终点
- 删除: fade-out
- 选中: 高亮闪烁效果
- 时长: 300ms ease-out
```

### 反馈动画

```
交互反馈:
- 点击: 轻微缩放效果
- 悬停: 平滑颜色过渡
- 拖拽: 实时位置跟随
- 错误: 轻微震动效果
```

## 实施指导

### 开发优先级

```
第一阶段 (核心功能):
1. 基础节点渲染
2. 简单连接线
3. 基础交互操作
4. 画布导航

第二阶段 (体验优化):
1. 动画效果
2. 性能优化
3. 响应式适配
4. 可访问性

第三阶段 (高级特性):
1. 主题切换
2. 复杂布局
3. 导出功能
4. 协作功能
```

### 质量标准

```
性能指标:
- 首屏加载: < 1.5秒
- 操作响应: < 50ms
- 60fps流畅度
- 内存使用: < 150MB

用户体验:
- 学习成本低
- 操作直观
- 错误恢复容易
- 跨平台一致
```
