| 文件名                     | 描述             | 主要内容                              |
| -------------------------- | ---------------- | ------------------------------------- |
| `architecture.mdc`         | 核心架构和技术栈 | Vue 3 + TypeScript, SPA设计, 目录结构 |
| `frontend-development.mdc` | 前端开发规范     | Vue 3规范, TypeScript规范, 代码标准   |

3. **项目信息** - 基本项目信息
